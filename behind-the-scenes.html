Behind the scenes

Things I used to create the website

    Introduction

    This is my website (go look at the rest of it if you haven't already, it was put at the end of the navbar for a reason). It was a fun process and I think it was very good for a first go at web development :)

    If you are curious about how I made any of this, the source code (all html, css and js files) is available on github.

    Libraries

    HTML/CSS/JS is too limited, so I had to use some external extensions in order to create this website, all of which are listed here:

    • jQuery: The go-to libarary for anything to do with DOM manipulation. It is used in every script I have written for the website, using functionalities from automating the table of contents generation to preserving light/dark theme preference through cookies.
    • Bootstrap: I don't know why Mr. Forbes didn't introduce us to it, but it has been a massive help to everything from containers to the navbar. It also made for an extremely responsive webpage that will function on phone screens just fine
    • SASS: I found this one while searching on YouTube once (all those google searches made it to my YouTube recommendations after all smh) and it made styling a lot easier. All the styling was done in .scss files and compiled to .css for the website to use.

    Potential Improvements

    While it is a complete standalone website, it isn't perfect, and here are some changes that could improve it:

    • Better accessibility (Alt text, ARIA, etc.)
    • Good CSS that doesn't depend on the HTML structure
    • Better index.html because it was kinda rushed ¯\_(ツ)_/¯
    • Light/dark theme diagrams and better theme colors overall
