Introduction

This website will teach the basics of communication technologies as seen in the IGCSE curriculum. It was created as an assignment in the course to assess both my communication technologies knowledge and my web development skills. Some of these bits include the different types of data transmission, how the Web works, the history of the Web, and me flexing my web development and jQuery skills more. Enjoy!

Note: Some features (such as the Table of Contents and sticky header) may not 'load' as I am using jQuery to automate them for me. I import jQuery from Google APIs which means if you are in China without a VPN, it will fail to load and the code will be unable to execute the tasks
Note 2: I have adjusted the page so that it uses a downloaded version of jQuery, which means while things may load a bit slower, it means you can see everything as normal :)
